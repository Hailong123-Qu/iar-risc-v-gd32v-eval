# SPI Flash example

For the __SPI Flash__ example, configure the Serial Port in the Virtual Terminal with the following settings:

| __Setting__    |  __Selection__  |
| :------------- | :-------------- |
| Baud rate      | 115200          |
| Data           | 8               |
| Bit Parity     | None            |
| Stop bit       | 1               | 
| Flow Control   | None            |
